# EIFER — 選手写真フォルダ

## ファイル命名規則
| ファイル名 | 配置場所 | 推奨写真 |
|---|---|---|
| `player1.jpg` | 左端（一番外側） | 守備・ピッチング系 |
| `player2.jpg` | 左寄り | 走塁・守備系 |
| `player3.jpg` | **中央メイン（最大・最前面）** | 打撃・ホームラン系 |
| `player4.jpg` | 右寄り | 走塁・守備系 |
| `player5.jpg` | 右端（一番外側） | ピッチング・守備系 |

## 写真仕様
- 形式：JPG または PNG（`.jpg` で統一推奨）
- サイズ：縦長（2:3 比率が理想）
- 解像度：最低 800×1200px 以上
- 人物が縦全身または上半身〜全身に見えるもの

## 注意
- Instagramのスクショ（UI込み）ではなく、元写真をダウンロードして入れること
- ファイルを入れると自動的にランディングページに反映される
