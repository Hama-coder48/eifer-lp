# EIFER — 選手写真フォルダ（7人構成）

## ファイル命名規則 & 配置
| ファイル名 | CSS クラス | 配置場所 | 推奨写真 |
|---|---|---|---|
| `player1.jpg` | `hp-l1` | 左端（一番外側・最小） | ピッチング・守備系 |
| `player2.jpg` | `hp-l2` | 左寄り | 走塁・守備系 |
| `player3.jpg` | `hp-l3` | 左内側 | バッティング・守備系 |
| `player4.jpg` | `hp-c` | **中央メイン（最大・最前面）** | ガッツポーズ・ホームラン系 |
| `player5.jpg` | `hp-r1` | 右内側 | バッティング・守備系 |
| `player6.jpg` | `hp-r2` | 右寄り | 走塁・守備系 |
| `player7.jpg` | `hp-r3` | 右端（一番外側・最小） | ピッチング・守備系 |

## 写真仕様
- 形式：JPG または PNG（`.jpg` で統一推奨）
- サイズ：縦長（2:3 比率が理想）
- 解像度：最低 800×1200px 以上
- 人物が縦全身または上半身〜全身に見えるもの

## レスポンシブ表示
| 画面幅 | 表示人数 |
|---|---|
| 1200px 以上 | 全7人 |
| 900〜1200px | 5人（l1・l3非表示） |
| 540〜900px | 3人（l1・l2・r2・r3非表示） |
| 540px 未満 | 中央1人のみ |

## 注意
- Instagramのスクショ（UI込み）ではなく、元写真をダウンロードして入れること
- ファイルを入れると自動的にランディングページに反映される（フェードインあり）
- 黒ユニフォーム写真が増えたら同じ命名でどんどん差し替え可能
